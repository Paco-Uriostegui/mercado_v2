feat	        Nueva funcionalidad	        feat: add user login
fix	        Corrección de bug	        fix: resolve crash on save
refactor	Cambios que no añaden ni corrigen	refactor: simplify auth logic
docs	        Documentación	                docs: update API README
chore	        Tareas de mantenimiento	        chore: update dependencies
test	        Pruebas	                        test: add unit tests for service
style	        Formato (sin cambio lógico)	style: fix code formatting
perf	        Mejora rendimiento	        perf: optimize image loading
